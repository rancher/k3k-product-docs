Muss mindestens 1 sein. Der Standardwert ist 1. + | 1 |

Dieses Feld ist unveränderlich. + |  | 
| *`clusterDNS`* __Zeichenfolge__ | ClusterDNS ist die IP-Adresse für den CoreDNS-Dienst. +

Dieses Feld ist unveränderlich. + |  |
